Bem-vindo ao Sistema de Gerenciamento de Eventos


Este sistema foi desenvolvido para gerenciar os diversos tipos de eventos, os usuários podem convidar outros usuários para diferentes tipos de eventos, o convite possui dados do evento como o nome do evento, local, a data


Através deste sistema, é possível organizar os eventos que o usuário foi convidado. Os dados asseguram que os usuários que foram convidados possam ter acesso as informações do evento.
